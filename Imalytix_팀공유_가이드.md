## 브라우저 익스텐션 설치 방법

### 준비물
- `imalytix-extension-v1.0.zip` 파일 (팀장에게 요청)
- Chrome 또는 Edge 브라우저

---

### Chrome 설치

1. `imalytix-extension-v1.0.zip` 압축 해제 → `dist` 폴더 확인
2. Chrome 주소창에 입력:
   ```
   chrome://extensions
   ```
3. 우측 상단 **개발자 모드** 토글 ON
4. **압축해제된 확장 프로그램 로드** 클릭
5. 압축 해제한 `dist` 폴더 선택
6. 설치 완료 ✅

---

### Edge 설치

1. `imalytix-extension-v1.0.zip` 압축 해제 → `dist` 폴더 확인
2. Edge 주소창에 입력:
   ```
   edge://extensions
   ```
3. 좌측 하단 **개발자 모드** 토글 ON
4. **압축된 파일 없이 로드** 클릭
5. 압축 해제한 `dist` 폴더 선택
6. 설치 완료 ✅

## 익스텐션 사용 방법

### 이미지 분석하기

1. 브라우저에서 분석하고 싶은 이미지가 있는 페이지 접속
2. 이미지 위에서 **우클릭**
3. **Imalytix로 분석** 선택
4. 우측 사이드패널에 분석 결과 표시

또는

1. 툴바의 **Imalytix 아이콘** 클릭 → 사이드패널 열기
2. 이미지 URL 직접 입력 후 분석

---

### 분석 결과 읽기

| 점수 | 의미 |
|------|------|
| 80점 이상 | AI 생성 가능성 매우 높음 🔴 |
| 60~79점 | AI 생성 의심 🟠 |
| 31~59점 | 판단 불확실 🟡 |
| 30점 이하 | 실제 이미지 가능성 높음 🟢 |

---

### 상세 분석 보기

사이드패널 결과 화면에서 **상세 분석 보기** 버튼 클릭
→ 웹 대시보드(imalytix.vercel.app)에서 의심 영역 시각화 확인 가능

---

## 웹 대시보드 사용 방법

1. https://imalytix.vercel.app 접속
2. 이미지 파일 업로드 또는 URL 입력
3. 분석 시작 → 결과 확인
4. 이미지 위 번호 클릭 → 의심 영역 상세 설명 확인

---

## 주의사항

- 백엔드 서버는 **15분 비활성 시 슬립** 상태로 전환됩니다
- 슬립 후 첫 요청은 **30~60초 로딩**이 발생할 수 있습니다 (이후 정상 속도)
- AI 판정은 참고용이며 100% 정확하지 않을 수 있습니다
