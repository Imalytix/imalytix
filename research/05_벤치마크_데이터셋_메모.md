# 벤치마크 데이터셋 메모

## 목적

Imalytix의 Phase A에서 현재 구현한 DINOv2 / CLIP baseline, source attribution, trust analysis가 실제로 얼마나 동작하는지 빠르게 검증하기 위한 로컬 벤치마크 메모입니다.

## 현재 데이터 구성

- 로컬 샘플 폴더: `samples_test/`
- 카테고리:
  - portraits
  - landscape
  - animals
  - food
  - architecture
  - abstract
  - products
  - text_ocr
  - charts
  - illustration

## 라벨링 방식

- `REAL`
- `AI`

## 현재 벤치마크 산출물

- manifest: `benchmarks/phase_a_manifest.json`
- report: `reports/phase_a_benchmark_report.md`

## 실행 결과 요약

- 총 이미지 수: 30
- DINO baseline label accuracy: 0.3667
- CLIP baseline label accuracy: 0.4667

## 해석

- 아직은 “정답률이 높은 분류기”라기보다 “구조를 검증하는 초기 파이프라인”에 가깝다.
- 이 단계에서는 정확도보다도 다음이 중요하다.
  - 결과 재현 가능성
  - 카테고리별 편향 확인
  - 유사도 검색이 정상 동작하는지
  - 로그와 리포트가 자동으로 남는지

## 다음 액션

- 실제 도용/재업로드 이미지와 합성/편집 이미지를 따로 모아 평가셋 확장
- 동일 카테고리 내 near-duplicate와 cross-category false positive 분리
- retrieval top-k의 설명 가능성까지 함께 검증
